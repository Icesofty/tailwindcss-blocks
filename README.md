# tailwindcss-blocks
MistralCSS provides free and pre-made TailwindCSS blocks to kickstart all your projects. 

## How To? 

- Choose a component @ https://mistralcss.com
- Choose a color Schema
- Copy / Paste the code
- Done. Yup, that's it.
